Portfolio Personal - Nathan Batallé

Este repositorio contiene mi portfolio profesional. Está diseñado para presentar mis proyectos de DAM de forma limpia y técnica.

🛠️ Implementación técnica
Aunque mi enfoque principal es el Backend, he aplicado los siguientes conceptos de Frontend:

Estructura: Uso de HTML5 semántico y Flexbox para el diseño de dos columnas.

Estilo: Diseño oscuro con efectos de transparencia (backdrop-filter) y variables CSS para la gestión de colores.

Interactividad: Animaciones suaves en las tarjetas de proyectos mediante transform y transition.

Adaptabilidad: Web Responsive mediante Media Queries para una visualización correcta en móviles.

📂 Estructura del proyecto
/index.html: Estructura principal de la web.

/styles/: Archivos CSS.

/img/: Recursos visuales.

/docs/: Recursos descargables.
